🍎 Установка на macOS

  1. Распакуй скачанный .tar.gz (двойной клик).
  2. Перетащи PsyGames.app в «Программы».
  3. Открой Терминал и выполни: xattr -cr /Applications/PsyGames.app
  4. Запусти приложение.
Команда нужна, потому что приложение пока без подписи Apple. На macOS Sonoma/Sequoia иначе пишет «повреждено» — это не вирус, а отсутствие подписи.

🪟 Установка на Windows

  1. Запусти скачанный PsyGames-windows-setup.exe.
  2. Если ругается SmartScreen — «Подробнее» → «Выполнить в любом случае».
  3. Пройди установщик. Приложение само обновляется при новых версиях.
Предупреждение SmartScreen — из-за отсутствия платной подписи Microsoft, не из-за вируса.

🤖 Установка на Android

  1. Скачай PsyGames-android.apk.
  2. Разреши «Установка из неизвестных источников» для браузера / файлового менеджера.
  3. Открой APK → «Установить».
