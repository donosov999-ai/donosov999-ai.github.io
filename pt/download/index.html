🍎 Instalar no macOS

  1. Descompacte o .tar.gz baixado (clique duplo).
  2. Mova o PsyGames.app para Aplicativos.
  3. Abra o Terminal e execute: xattr -cr /Applications/PsyGames.app
  4. Abra o aplicativo.
O comando é necessário porque o app ainda não tem assinatura da Apple. No macOS Sonoma/Sequoia ele aparece como "danificado" — não é vírus, apenas falta a assinatura.

🪟 Instalar no Windows

  1. Execute o PsyGames-windows-setup.exe baixado.
  2. Se o SmartScreen avisar — clique em "Mais informações" → "Executar assim mesmo".
  3. Conclua o instalador. O app se atualiza sozinho a cada nova versão.
O aviso do SmartScreen ocorre por falta da assinatura paga da Microsoft, não por malware.

🤖 Instalar no Android

  1. Baixe o PsyGames-android.apk.
  2. Permita "Instalar de fontes desconhecidas" para o seu navegador / gerenciador de arquivos.
  3. Abra o APK → "Instalar".
