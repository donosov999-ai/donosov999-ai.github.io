记忆矩阵

记住并重现亮起的方格

关于这个训练

《记忆矩阵》会展示一个方格，其中一些格子短暂地亮起，随后归于空白。你的任务是凭记忆复现这一图案，准确地点按那些曾经亮过的格子。随着进程推进，方格会变大、图案会变密，于是每一轮都要求你一次记住稍多一点的空间信息。

锻炼什么

它训练视觉空间工作记忆和短时空间回忆，即把若干条目“在哪里”这一信息在脑中保持几秒钟的能力。它也依赖专注的注意力，因为在图案闪现时一旦走神，整轮就泡汤了。

历史

这项练习是神经心理学中经典空间广度测验的数字后裔。这种方格-闪现的形式在 21 世纪头十年末和 2010 年代被脑力训练应用推广给大众，其中 Lumosity 的 Memory Matrix 是最知名的版本，也是这个名字的出处。

由谁、何时创建

它没有单一的发明者。这一形式直接源自科西方块敲击测验，由加拿大-意大利籍心理学家菲利普·科西（Philip Corsi）于 1972 至 1973 年前后设计，作为数字广度的空间对应物，并承接 D. O. Hebb 早先的工作。多数人所熟知的那款精致方格游戏，是由 Lumos Labs（Lumosity）打造并命名为 Memory Matrix 的。

如何训练

别一个一个地去记格子。把亮起的格子归并成形状、线条和簇团，并留意对角线或对称。让视线停在中心附近，以便一眼看到整个方格；与其数方格，不如默默记下大致的位置（左上角、中间一行）。把难度往上推，直到你大约三分之一的次数会失败——那个边界，才是记忆广度真正得到拉伸的地方。

练习多久

短时段最有效：五到十分钟、每周几次、全神贯注，胜过漫长的马拉松。空间记忆很快就会疲劳，所以一旦你的准确率明显下滑，就停下，而不要硬撑着熬过疲劳。

研究依据

证据最充分的恰恰是你能预料到的部分——你会更擅长记忆方格图案，以及类似的空间广度任务。更宽泛的承诺则要薄弱得多：工作记忆训练的元分析，包括 Melby-Lervåg 和 Hulme 那篇广受引用的 2016 年综述，几乎找不到可靠的、向整体智力、学业表现或日常记忆的远迁移。Lumosity 自己也曾在 2016 年因缺乏依据的脑力训练宣传，向美国联邦贸易委员会支付了 200 万美元和解金，所以对任何“提升智商”或“预防衰退”的说辞，都要抱以真正的谨慎。

建议

把它当作磨砺空间注意力的专注热身，而非健忘症的解药；以你自己的广度在数周内的增长来评判进步，而不是以那些关于“活得更聪明”的说法。

常见问题

记忆矩阵会让我现实生活中的记忆变好吗？

它能可靠地提升你在这类及类似方格任务上的表现，但研究并不支持它能广泛地提升日常记忆或智力。请把它当作针对空间回忆的专项锻炼，而非一次全面升级。

这和普通的翻牌配对记忆游戏有什么不同？

配对游戏（如“专注力”翻牌）是通过多次翻动、反复试错来考验“哪个符号在哪里”。记忆矩阵则把完整图案展示一次，再要求你凭这一眼短暂的观看复现其确切位置，这更直接地考验空间工作记忆。

拿高分最好的办法是什么？

把亮起的格子归并成形状和线条，而不是去记单个方格；让视线停在中心附近，以便把整个方格尽收眼底；并停留在一个让你失败得足够频繁、因而具有挑战的难度上。轻松过关的回合几乎练不到什么。

变体

常见的变体会改变方格大小（从小的 3×3 到大型棋盘），会依次展示数个图案，会要求你复现格子亮起的顺序而不仅是那一组，会加入干扰性闪烁，或对你的作答设置时间限制。某些版本会让图案淡出或移动，以加大回忆难度。
