SET：三元组

按 4 种属性找出三元组

关于这个训练

「SET 三连」是一款视觉规律游戏，它基于一副每张牌都在四个特征（数量、形状、颜色和填充）上各有变化、每个特征又各有三种取值的牌组。你的任务是扫视桌面上的牌，找出一组有效的「三连」：对于这四个特征中的每一个，三张牌要么全都相同，要么全都不同，绝不允许出现「两张一样、一张不同」的混搭。

锻炼什么

它在时间压力下训练视觉搜索和特征绑定，外加工作记忆中「同时玩转多条规则」的那部分——因为你要在脑中记住四个条件，并一次性在三张牌上逐一核对它们。它同样依赖认知灵活性，因为你得不断放弃一组候选三连，转而去扫视新的一组。

历史

这款游戏源自一位遗传学家用来记录数据中出现了哪些性状的速记法，后来在实验室里和朋友之间变成了一款纸牌游戏。它于1991年商业出版，同年赢得了美国门萨的「Mensa Select」大奖，此后成了数学课堂和谜题圈里的常客——部分原因在于它的结构恰好能干净利落地对应到有限几何上。

由谁、何时创建

由遗传学家玛莎·琼·法尔科（Marsha Jean Falco）于1974年创作，当时她正在英国剑桥研究遗传学，并于1991年由她的公司 Set Enterprises 出版。所以，与许多抽象谜题不同，SET 有一位单一的、记载翔实的发明者。

如何训练

挑一个特征，比方说颜色，在动手找之前先在脑中把桌面分成若干组，而不是去比对随机的牌对。要善用这条规则：任意两张牌都能唯一确定第三张能凑成三连的牌，所以一旦有两张牌抓住了你的眼球，你只需要想象并找出那一个「搭档」即可。强迫自己在确定之前核对全部四个特征，因为一次快速的误判通常都是漏看了填充。

练习多久

短而频繁的回合效果最好：每次五到十五分钟，每周练几次。当你搜索变慢、错误开始悄悄冒头时就停下来，因为带着疲劳硬磕，磨练出的多半是挫败感，而不是速度。

研究依据

证据最扎实的，恰恰是你能预料到的那一点：你会更快、更准地找出三连，也更擅长这种多特征的视觉搜索。至于说它能提升一般智力、学校数学成绩或日常推理，这类说法则要薄弱得多，这与更广泛的认知训练文献相符——其中近迁移效果有限，远迁移则基本未获证实，所以对那些宏大的承诺要保持谨慎。

建议

以短时段冲刺的方式来玩，并坚持在每一组三连上核对全部四个特征，而不只是那两个一眼跳出来的特征——准确优先，速度自然会跟上来。

常见问题

到底什么才算一组「三连」？

三张牌，其中数量、形状、颜色和填充这四个特征里的每一个，要么在三张上完全相同，要么在三张上各不相同。只要有任何一个特征出现「两张一种、一张另一种」的情况，它就不是一组三连。

这会让我变聪明，或者数学变好吗？

老实说，它主要会让你更擅长玩 SET 和快速视觉搜索。没有确凿的证据表明它能提升一般智力或数学成绩，不过它确实是一种很好的、针对发现规律的专注训练，而且它与有限几何契合得相当漂亮。

我会僵住，什么都找不到，怎么办？

别一下子去扫所有的牌。盯住一个特征，按它把牌分组，然后利用「任意两张牌都唯一指向能凑成三连的那一张牌」这一点，只去找那一张。

变体

常见的变体包括：与时间赛跑的单人限时模式、「谁先喊出来谁赢」的快节奏多人竞赛、去掉填充、只剩三个特征的简化牌组，以及一些谜题模式——它们会问在某个固定布局中存在多少组三连，或保证其中至少有一组。
