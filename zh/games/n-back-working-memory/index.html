N-back：工作记忆

当前闪烁的位置是否与 N 步之前的相同？

关于这个训练

刺激一个接一个地依次出现；每当当前刺激与 N 步之前出现过的那一个相同时，你就作出反应。随着 N 从 1 升到 2、再到 3，工作记忆的负荷陡然加重。

锻炼什么

工作记忆容量、更新能力（不断地把旧条目换成新条目），以及抗干扰能力——这正是专注和推理背后核心的“心理杂耍”。

历史

N-back 流程于 1958 年作为一种工作记忆的实验室测量手段被引入。它在 2008 年声名大噪，当时一项研究报告称，训练更难的“双重”版本能提升流体智力——由此点燃了现代脑力训练热潮和长达十年的争论。

由谁、何时创建

这项任务归功于 Wayne Kirchner（1958）。那个颇具影响力的训练论断则出自 Susanne Jaeggi、Martin Buschkuehl 及其同事（2008）。

如何训练

选一个你能维持在约 80% 准确率的 N，并让它自适应——准的时候往上加，失手的时候往下降。先训练单一通道；只有当你觉得单一 n-back 已能掌控时，再加入双重（位置 + 声音）版本。

练习多久

训练研究采用的是大约每天 15 到 20 分钟、持续数周。如果你想给迁移留一丝可能，关键变量正是这种持续性，而非偶尔的一次长时段练习。

研究依据

对争议要坦诚。2008 年的结果令人振奋，但若干设有严格对照的重复实验（如 Redick 2013）和元分析发现，在健康成年人身上几乎没有、甚至完全没有向流体智力的迁移。可靠的是：你在 n-back 及与之密切相关的任务上会稳定进步。把“提升智商”当作未经证实即可。

建议

为专注和更新能力而练它，为打破自己 N 值纪录的成就感而练它——而不是冲着“智商更高”的承诺。让它保持自适应，并在疲劳毁掉你的准确率之前就停下。

常见问题

N-back 能提升智商吗？

那项著名的 2008 年研究说有可能；后来的重复实验大多未能证实。稳妥的说法是：你会更擅长 n-back 和工作记忆类任务。

单重还是双重 n-back？

从单重开始。双重（位置 + 声音）更难，也正是最初那些训练研究所采用的版本。

多久才能见到进步？

任务本身的进步通常几天内就会显现；至于任何更广泛的迁移，若真会发生，也需要数周的每日练习。

变体

单重 n-back（单一刺激流）；双重 n-back（视觉 + 听觉）；以及位置、字母、颜色或声音等变体。难度通过提高 N 来递增。
