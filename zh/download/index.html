🍎 在 macOS 上安装

  1. 解压下载的 .tar.gz（双击）。
  2. 将 PsyGames.app 移到「应用程序」。
  3. 打开「终端」并运行：xattr -cr /Applications/PsyGames.app
  4. 启动应用。
该命令是必要的，因为应用尚未经过 Apple 签名。否则在 macOS Sonoma/Sequoia 上会显示「已损坏」——这不是病毒，只是缺少签名。

🪟 在 Windows 上安装

  1. 运行下载的 PsyGames-windows-setup.exe。
  2. 若 SmartScreen 提示——点击「更多信息」→「仍要运行」。
  3. 完成安装。新版本发布时应用会自动更新。
SmartScreen 提示是因为缺少付费的 Microsoft 签名，并非恶意软件。

🤖 在 Android 上安装

  1. 下载 PsyGames-android.apk。
  2. 为浏览器 / 文件管理器允许「安装未知来源应用」。
  3. 打开 APK →「安装」。
