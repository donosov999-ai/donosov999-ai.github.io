🍎 Install on macOS

  1. Unpack the downloaded .tar.gz (double-click).
  2. Move PsyGames.app to Applications.
  3. Open Terminal and run: xattr -cr /Applications/PsyGames.app
  4. Launch the app.
The command is needed because the app is not yet Apple-signed. On macOS Sonoma/Sequoia it otherwise shows "damaged" — it is not a virus, just a missing signature.

🪟 Install on Windows

  1. Run the downloaded PsyGames-windows-setup.exe.
  2. If SmartScreen warns — click "More info" → "Run anyway".
  3. Finish the installer. The app auto-updates on new releases.
The SmartScreen warning is due to the missing paid Microsoft signature, not malware.

🤖 Install on Android

  1. Download PsyGames-android.apk.
  2. Allow "Install from unknown sources" for your browser / file manager.
  3. Open the APK → "Install".
