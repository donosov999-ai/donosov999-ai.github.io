🍎 Installation unter macOS

  1. Entpacke die heruntergeladene .tar.gz (Doppelklick).
  2. Verschiebe PsyGames.app in „Programme".
  3. Öffne das Terminal und führe aus: xattr -cr /Applications/PsyGames.app
  4. Starte die App.
Der Befehl ist nötig, weil die App noch nicht von Apple signiert ist. Unter macOS Sonoma/Sequoia erscheint sonst „beschädigt" — kein Virus, nur fehlende Signatur.

🪟 Installation unter Windows

  1. Starte die heruntergeladene PsyGames-windows-setup.exe.
  2. Wenn SmartScreen warnt — „Weitere Informationen" → „Trotzdem ausführen".
  3. Schließe den Installer ab. Die App aktualisiert sich bei neuen Versionen selbst.
Die SmartScreen-Warnung liegt an der fehlenden kostenpflichtigen Microsoft-Signatur, nicht an Schadsoftware.

🤖 Installation unter Android

  1. Lade PsyGames-android.apk herunter.
  2. Erlaube „Installation aus unbekannten Quellen" für deinen Browser / Dateimanager.
  3. Öffne die APK → „Installieren".
